First Bank Payment (USSD)

Read Instructions if you want to make contribution using First Bank
  • Use the phone number registered on your First Bank Account details for First Bank Account holders.
  • Dial *894*MERCHANT CODE*AMOUNT# or click the button labelled First Bank
  • Select the funding account number
  • Input your registered phone number with the PFA
  • Input your 5-digit ATM PIN

Other Banks Users Payment (USSD)

Read Instructions if you want to make contribution using Other Banks
  • For other banks account holders; Dial *402*MERCHANT CODE*AMOUNT# and processed with selecting your desired bank to transact the same.
  • Then select the funding account for the transaction
  • And input your registered phone number with the PFA
  • Input your 5-digit ATM PIN
